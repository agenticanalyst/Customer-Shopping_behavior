# 📊 Customer Shopping Behavior Dashboard | Tableau

## 📌 Overview

The Customer Shopping Behavior Dashboard is an interactive Tableau project that analyzes customer purchasing patterns, product performance, payment preferences, and sales trends. The dashboard helps businesses understand customer behavior and make data-driven decisions through interactive visualizations.

# 💡 Key Insights

- Clothing generated the highest sales.
- Young adults contributed significantly to overall revenue.
- Credit Card and UPI were the most preferred payment methods.
- Monthly sales trends helped identify purchasing patterns.
- Brand-wise analysis highlighted top-performing brands.
